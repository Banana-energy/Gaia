### 🍋 介绍

一个网页版SSH端和SFTP文件管理，用到的库包括socket.io, xterm.js, echarts 和 express

通过Echarts可视化机器CPU和内存负载情况

基于Vue实现了文件管理功能，包括上传、下载、删除和编辑文件权限

### 🍦 开始

```shell
git clone https://github.com/Banana-energy/Gaia.git

npm install

npm run server

npm run start
```
